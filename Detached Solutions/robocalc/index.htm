About RoboCalc

The RoboCalc project is designed to lower the learning curve required to operate and program the Norland Research Robot. This program is a great addition to any classroom utilizing the robot! Students can easily control the robot using easy to remember mnemonics in the format XLR where x is the command (T for timed, S for Switch and TS for Time and Switch) R is the right wheel direction and L is the left wheel direction. For example, students can write 'sfn' for "Switch" Forward, Neutral which makes the robot pivot on the right wheel while turning left. This is much simpler than trying to remember numbers and special commands such as "Send {221}:Get t"
